== Installation ==

1. Upload `rocketfuel-cache` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! RocketFuel automatically configures optimal settings for your server

= Manual Installation =

1. Download the plugin ZIP file
2. Go to WordPress Admin > Plugins > Add New > Upload Plugin
3. Choose the ZIP file and click Install Now
4. Activate the plugin

== Frequently Asked Questions ==

= Does RocketFuel work with WooCommerce? =

Yes! RocketFuel automatically excludes dynamic WooCommerce pages (cart, checkout, my account) from caching. Pro version adds WooCommerce-specific optimizations like cart fragment control and selective script loading.

= Can I use RocketFuel with other caching plugins? =

We recommend using only RocketFuel for caching to avoid conflicts. RocketFuel will detect other caching plugins and warn you if conflicts are found.

= Will RocketFuel break my site? =

RocketFuel includes a Safe Mode toggle that disables all optimizations with one click. If anything looks wrong, enable Safe Mode, adjust settings, and re-enable.

= Do I need the Pro version? =

The free version includes 89 features that beat most paid caching plugins. Pro adds image optimization, critical CSS, script manager, and advanced features for users who want maximum performance.

= How does the 15-day trial work? =

Enter your email in the License tab, click "Activate Trial", and all Pro features are instantly unlocked for 15 days. No credit card needed. After the trial, your site gracefully returns to the free version with zero downtime.

= Is RocketFuel compatible with page builders? =

Yes. RocketFuel works with Elementor, WPBakery, Divi, Beaver Builder, Gutenberg, and all major page builders.
